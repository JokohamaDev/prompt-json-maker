# Prompt JSON Maker

A clean, minimal web app for building structured prompts for AI image generation.

Describe your subject, scene, lighting, and camera settings through an intuitive form — and get a standardized JSON output ready to paste into Nano Banana, Seedream v4, Grok Imagine, or any AI image generator that accepts structured prompts.

**[Try it live](https://jokohamadev.github.io/prompt-json-maker/)**

## Features

- **Real-time JSON preview** as you fill in the form
- **One-click copy** to clipboard
- **Random Selector** for quick inspiration
- **Save & Load** configurations across sessions
- **Responsive** — works on desktop and mobile

## Getting Started

Clone the repo and open `index.html` in your browser. No build step required.

```bash
git clone https://github.com/JokohamaDev/prompt-json-maker.git
cd prompt-json-maker
open index.html
```

## Supported Platforms

- Nano Banana
- Seedream v4
- Grok Imagine
- Any AI image generator accepting structured JSON prompts
